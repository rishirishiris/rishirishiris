Blog 1: Beta testing available soon!

By: Hristo (hristo@inflightmadness.com) on September 2nd, 2024

Hello everyone! We have added a brand-new blog section to the IFM website to convey information about the game's development and release easier. Speaking of which, we are pleased to announce that development has reached an advanced-enough stage that we are almost ready to start a beta testing process for our game. During the beta testing process, testers are expected to give feedback to us, the developers, and tell us what we should fix or change. More details about the release of the program and the beta testing application process will be revealed soon in another blog post, so be sure to check this page frequently for updates.
